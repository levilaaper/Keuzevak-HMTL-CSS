Reflectie

Ik heb eerder twee jaar lang Creative Media and Gaming Technologies gestudeerd. Hier ben ik al in aanraking gekomen met het schrijven van code. Hierdoor wist ik al de basis beginselen van HTML en CSS.

Dit keuzevak heeft er voor mij weer eens voor gezorgd dat ik met de hand code ging schrijven en structureren. Hierdoor ben ik weer gemotiveerd geworden om vaker HTML en CSS te gaan schrijven. Ik heb dus niet echt nieuwe specifieke kennis opgedaan maar heb wel een refresh gehad van de dingen die ik al wist.
